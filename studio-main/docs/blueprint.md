# **App Name**: Innovators Hub

## Core Features:

- Dynamic Content Loading: Efficiently load and display project data, team member profiles, event schedules, and gallery images as users navigate the single-page application.
- Interactive Map: Integrate a dynamic map using MapLibre to display the club's location, allowing users to easily find and interact with the club's physical address.
- Subscription Manager: Implement a Newsletter subscription function for keeping in touch with visitors of the web site.
- Glassmorphism UI Components: Create reusable, translucent UI components with blurred backgrounds and subtle light borders for cards, forms, and other interactive elements.
- Form Submission Handling: Collect all form data for contact forms, and save them in a database for retrieval.
- AI content generation: If any generated content has need of images, this tool assists the site administrator by requesting images corresponding to some text via an external API. The site admin chooses whether to display any suggested images. The AI functions only as an image suggestion tool.

## Style Guidelines:

- Background color: Dark charcoal (#121212) to set a modern, dark mode tone.
- Primary color: Electric Purple (#BE68FA) to add a vibrant, innovative feel.
- Accent color: Cyan (#64FFDA) for highlights, links, and interactive elements.
- Body and headline font: 'Poppins', a modern sans-serif for clean, contemporary text. Note: currently only Google Fonts are supported.
- Use Lucide React icons throughout the website, ensuring consistent style and clarity.
- Utilize Tailwind CSS grid for Bento Grid layouts in the 'About' section to provide an interesting and balanced structure.
- Implement Framer Motion for 'fade-up' animations and hover effects to add smooth, engaging interactions on scroll.
